README

The inspiration for this tie fighter was taken from the image included: "tiefighter.gif".
To run the program, go through the directory in the terminal, compile Main.java with "javac Main.java",
and run with "java Main".


Controls:

  Rotation: 'E' and 'D'
  Scale: 'R' and 'F'
  Translation: Up key
